```
┌──────────────────────────────────────────────────────┐

│                                                        │

│  → Research questions                                  │
│                                                        │

│  → "Let me walk you through our methodology."          │
│                                                        │
│  [5:00–8:00] FINDINGS & DISCUSSION                     │
│  → Key results with data                               │
│  → Interpretation                                      │
│  → "As you can see from this graph..."                 │
│                                                        │
│  [8:00–9:00] IMPLICATIONS & LIMITATIONS                │
│  → Practical implications                              │
│  → Acknowledged limitations                            │

│                                                        │

│  → Main takeaway                                       │

│                                                        │
│  [10:00–13:00] Q&A SESSION                             │

│  → "That's a great question..."                        │
│  → "Based on our data..."                              │
│                                                        │
└──────────────────────────────────────────────────────┘
```

| Tiêu chí | Yếu (1-3) | Trung bình (4-6) | Khá (7-8) | Xuất sắc (9-10) |
|----------|-----------|-------------------|-----------|-----------------|
| **Content** | Không rõ ràng | Cơ bản, thiếu chi tiết | Rõ ràng, đầy đủ | Sâu sắc, hấp dẫn |
| **Organization** | Không có cấu trúc | Cấu trúc cơ bản | Tổ chức tốt | Hoàn hảo, signposting rõ |
| **Language** | Từ vựng đơn giản | Từ vựng trung bình | Academic language tốt | Phong phú, chính xác |
| **Fluency** | Ngắt quãng nhiều | Nói được, hay dừng | Trôi chảy, ít dừng | Hoàn toàn tự nhiên |
| **Pronunciation** | Khó hiểu | Hiểu được nhưng nhiều lỗi | Rõ ràng | Chuẩn, tự nhiên |
| **Q&A Handling** | Không trả lời được | Trả lời cơ bản | Trả lời tốt | Xuất sắc, insightful |
| **Visual Aids** | Không có | Đơn giản | Hỗ trợ tốt | Professional, effective |
| **Professional Presence** | Thiếu tự tin | Tự tin cơ bản | Tự tin, engaging | Commanding, inspiring |

### Bài thi 2: MOCK ENGLISH TEACHING SESSION (10 phút)

```
┌──────────────────────────────────────────────────────┐
│  MOCK ENGLISH TEACHING SESSION                         │
│  Thời lượng: 10 phút                                   │
├──────────────────────────────────────────────────────┤
│                                                        │
│  [0:00–1:30] WARM-UP & INTRODUCTION                    │
│  → Greeting students                                   │
│  → Warm-up question                                    │
│  → Learning objectives                                 │
│  → "Good morning, class! Today we're going to learn..."│
│                                                        │
│  [1:30–4:00] CONTENT EXPLANATION                       │
│  → Define concept                                      │
│  → Give examples                                       │
│  → Use analogy                                         │
│  → Check understanding                                 │
│  → "Think of it like..."                               │
│  → "Does that make sense?"                             │
│                                                        │
│  [4:00–7:00] STUDENT ACTIVITY                          │
│  → Pair/Group discussion                               │
│  → Teacher circulates                                  │
│  → Facilitates discussion                              │
│  → "Discuss with your partner for 3 minutes."          │
│                                                        │
│  [7:00–9:00] DISCUSSION & FEEDBACK                     │
│  → Students share                                      │
│  → Teacher gives feedback                              │
│  → Builds on student responses                         │
│  → "Excellent observation!"                            │
│                                                        │
│  [9:00–10:00] REVIEW & CLOSING                         │
│  → Summary of key points                               │
│  → Homework assignment                                 │
│  → "The key takeaway today is..."                      │
│                                                        │
└──────────────────────────────────────────────────────┘
```

#### Evaluation Criteria — Teaching

| Tiêu chí | Yếu (1-3) | Trung bình (4-6) | Khá (7-8) | Xuất sắc (9-10) |
|----------|-----------|-------------------|-----------|-----------------|
| **Opening** | Không greeting | Greeting cơ bản | Tự tin, engaging | Năng lượng cao, hook hay |
| **Explanation** | Không rõ ràng | Cơ bản, thiếu ví dụ | Rõ ràng, có ví dụ | Sáng tạo, dễ hiểu |
| **Student Engagement** | Không tương tác | Tương tác cơ bản | Tạo discussion tốt | Highly interactive |
| **Classroom Language** | Không sử dụng | Câu cơ bản | Tự nhiên | Phong phú, professional |
| **Feedback** | Không đưa được | Feedback cơ bản | Constructive, specific | Inspiring, motivating |
| **Question Quality** | Câu hỏi đơn giản | Câu hỏi trung bình | Nhiều cấp Bloom's | Powerful, thought-provoking |
| **Time Management** | Không kiểm soát | Quản lý cơ bản | Quản lý tốt | Perfect pacing |
| **Closing** | Không tổng kết | Tổng kết cơ bản | Rõ ràng, homework | Memorable, inspiring |

### Bài thi 3: MOCK COUNSELING SESSION (10 phút)

```
┌──────────────────────────────────────────────────────┐
│  MOCK SCHOOL COUNSELING SESSION                        │
│  Thời lượng: 10 phút                                   │

│                                                        │
│  [0:00–2:00] RAPPORT BUILDING                          │
│  → Welcome + Safe space                                │

│  [2:00–4:00] ISSUE EXPLORATION                         │

│                                                        │
│  [6:00–8:00] STRATEGY DEVELOPMENT                      │
│  → "What has helped in the past?"                      │
│  → Collaborative problem-solving                       │
│  → Create action plan                                  │
│                                                        │
│  [8:00–9:30] SUPPORT & RESOURCES                       │
│  → Safety plan (if needed)                             │
│  → Referral (if appropriate)                           │
│  → "You're not alone."                                 │
│                                                        │
│  [9:30–10:00] CLOSING                                  │
│  → Follow-up plan                                      │
│  → "I'm here whenever you need."                       │
│  → Encouragement                                       │
│                                                        │
└──────────────────────────────────────────────────────┘
```

#### Evaluation Criteria — Counseling

| Tiêu chí | Yếu (1-3) | Trung bình (4-6) | Khá (7-8) | Xuất sắc (9-10) |

| **Empathy** | Không thể hiện | Đồng cảm cơ bản | Tự nhiên, chân thành | Deeply empathetic |
| **Open Questions** | Không đặt được | Câu hỏi cơ bản | Câu hỏi sâu | Powerful questioning |
| **Boundaries** | Không hiểu | Hiểu cơ bản | Giữ tốt | Professional mastery |
| **Solution Focus** | Không hướng giải pháp | Đề xuất cơ bản | Develop strategies | Outstanding problem-solving |
| **Sensitive Topics** | Không xử lý được | Xử lý cơ bản | Xử lý tốt | Expert handling |
| **Closing** | Không follow-up | Follow-up cơ bản | Follow-up tốt | Warm, comprehensive |

### 💡 FINAL TIPS

```
🌟 Bạn đã đi một chặng đường dài!
   → Từ "I can't speak English" 
   → Đến "I can teach, counsel, and present in English"
   → Tự hào về bản thân!

🌟 English là công cụ, không phải mục tiêu
   → Mục tiêu: Kết nối với thế giới
   → Chia sẻ kiến thức
   → Hỗ trợ học sinh tốt hơn

🌟 Sai lầm = Học hỏi
   → Mỗi lần sai = 1 lần học
   → Người bản xứ cũng sai
   → Quan trọng: Truyền đạt được ý

🌟 Tiếp tục practice mỗi ngày
   → 15 phút/ngày > 2 giờ/tuần
   → Consistency > Intensity
   → Mỗi ngày một chút = tiến bộ rõ rệt

🌟 Chia sẻ kiến thức
   → Dạy lại cho đồng nghiệp
   → Tạo study group
   → Học cùng = học nhanh hơn

🌟 Bạn ĐÃ SÀN SÀNG cho môi trường quốc tế!
   → Tự tin bước ra thế giới
   → Nghiên cứu của bạn xứng đáng được chia sẻ
   → Học sinh của bạn xứng đáng có giáo viên quốc tế

   🎉 CONGRATULATIONS! 🎉
```
